# 常用快捷键

以下是常见快捷键

| 操作 | Windows 快捷键 | macOS 快捷键 | 备注 |
| --- | --- | --- | --- |
| 放大 | Ctrl = | ⌘ = | 即键盘上的 `+` 号键位 |
| 缩小 | Ctrl - | ⌘ - |  |
| 原尺寸 | Ctrl 1 | ⌘ 1 | |
| 适应视图 | Ctrl 0 | ⌘ 0 | |
| 过滤画板 | Ctrl E | ⌘ E | 激活左边栏搜索画板功能 |
| 退出过滤画板 | ESC | ESC | 当前处于过滤状态 |
| 全选 | Ctrl A | ⌘ A |  |
| 切换视图 | tab | tab | 隐藏边栏-左侧边栏切换 |
